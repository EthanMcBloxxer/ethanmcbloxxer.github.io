Most people hate on Edge for a number of reasons, but is it actually bad?

Well, my answer to this question is no. Edge is introducing more features, better user interface, and even much better browsing power by switching to a revision of Chromium (the open-source project that powers Chrome). Microsoft even tries to make it very easy for people to switch from web browsers like chrome, because they can merge favorite sites, passwords, and more from Chrome, Firefox, IE, older versions of Edge, and even HTML files of favorites, which means that you can import these from almost any web browser.
